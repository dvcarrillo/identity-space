About identity.space

identity.space is a multimodal experience that builds generative visual and audio scapes from your own identity.

Connection

The phone application needs to communicate with the identity.space web server. Make sure that its address is correctly set on the Settings tab.

About us

identity.space has been built as a project for the KTH DT2140 (Multimodal Interaction and Interfaces) course at KTH Royal Institute of Technology on November 2020. This project is open source, find it on GitHub.
